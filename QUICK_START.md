# ColCord Careers - Quick Start

## 🚀 Quick Setup

1. **Install dependencies:**
   ```bash
   npm install
   cd server && npm install && cd ..
   ```

2. **Configure Google Apps Script** (see code below)

3. **Start development servers:**
   ```bash
   # Terminal 1: Frontend
   npm run dev
   
   # Terminal 2: Backend
   npm run server:dev
   ```

4. **Open your browser:**
   - Frontend: http://localhost:5173
   - API: http://localhost:3001
